# Used-Car-Sales-Inventory

Used-Car-Sales-Inventory is a simple program that helps to do basic inventory management for a car dealership manager.

#Getting Started

After cloning the repo:
### Install the gems

```
bundle install
```

### Setup the database

Create the database
```
rake db:create
```

Migrate the database
```
rake db:migrate
```

### Start the server
```
rails s
```
